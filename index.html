Este app só roda no celular

E o motivo é puramente comportamental:

  • Micro-Moments: Dados do Google mostram que o celular é o vilão usado para preencher micro-vácuos de tempo (esperando um café, no elevador ou no comercial da TV). O desktop é para ações intencionais. Você já abriu um notebook só para ficar scrollando à toa?
  • Formato de Feed: Relatórios globais (Statcounter e DataReportal) comprovam que feeds de consumo rápido têm até 90% do engajamento vindo de dispositivos móveis.

Este site foi desenhado para hackear o comportamento nativo de scrolling móvel (o famoso doomscrolling). Por isso, não disponibilizamos a versão desktop.
